# MP Preview

一键将 Obsidian 笔记转换为微信公众号格式的插件。

![version](https://img.shields.io/github/v/tag/rezonegame/mp-preview?color=blue&label=version&style=flat) ![license](https://img.shields.io/badge/license-MIT-green)

## ✨ 功能特点

### 核心功能
- 📝 一键将 Markdown 文档转换为微信公众号格式
- 🎨 多种精美模板（Minimal、Focus、Elegant、Bold 等系列），支持自定义字体和字号
- 🖼️ 自定义背景和样式，实时预览编辑效果
- 📋 一键 **Pub 复制**，直接粘贴到微信公众号编辑器
- 📸 支持导出为长图
- 🔒 锁定预览，避免编辑期间刷新打断

### v1.8 新增功能

#### v1.8.4 (2026-03-10)
- ✨ **新增：预览界面一键刷新按钮** — 在顶部辅助工具栏新增刷新按钮，点击可随时重新渲染预览内容，不受锁定状态限制。

#### v1.8.3 及之前版本
- 🔍 **SEO 隐藏文字** — 通过 Obsidian 弹窗插入不可见但可被搜索引擎索引的关键词
- 🎯 **Callout 支持** — 完整支持 18 种 Obsidian Callout 类型（`> [!tip]`、`> [!warning]` 等），自动转换为带颜色色条和 emoji 图标的美化引用块
- 📋 **FrontMatter 标题卡片** — 自动解析 YAML 头中的 `title` / `author` / `date`，生成渐变标题卡片（可在设置中开启）
- 🌙 **深色模式隔离** — 预览区始终保持浅色显示，不受 Obsidian 深色主题影响
- ✍️ **中文排版优化** — 基于 pangu.js 自动在中英文之间添加空格，支持智能引号转换（`「」` ↔ `""`）
- 📜 **滚动位置保持** — 使用百分比方案精确恢复预览滚动位置
- 🖼️ **图片注释编辑** — 点击预览中的图片即可编辑 Alt Text / 图注

## 📦 安装方法

### 方式一：从 Obsidian 社区插件安装
1. 打开 Obsidian → 设置 → 第三方插件
2. 关闭安全模式
3. 点击「浏览社区插件」→ 搜索 **MP Preview**
4. 点击安装并启用

### 方式二：通过 BRAT 安装（推荐，获取最新版本）
1. 先安装社区插件中的 [BRAT](https://github.com/TfTHacker/obsidian42-brat) 插件
2. 在 BRAT 设置中添加仓库：`https://github.com/rezonegame/mp-preview`
3. BRAT 会自动检测新版本并提示更新

### 方式三：手动安装
1. 从 [Releases](https://github.com/rezonegame/mp-preview/releases) 下载 `main.js`、`manifest.json`、`styles.css`
2. 复制到 Obsidian Vault 的 `.obsidian/plugins/mp-preview/` 目录
3. 重启 Obsidian 并启用插件

## 💡 使用技巧

- 使用 🔒 **锁定**按钮可暂停实时预览，避免编辑大文档时频繁刷新
- ✏️ **编辑模式**允许在预览中直接修改文字（仅影响复制内容，不会修改 Markdown 源文件）
- 🔍 **SEO 隐藏文字**在预览中显示为淡色提示框，复制到公众号后完全不可见但可被搜索引擎索引
- 不同系列的模板适合不同风格的文章，可以通过「系列」筛选器快速切换
- 支持代码块、表格、图片、Callout 等 Markdown 元素
- Callout 语法示例：`> [!tip] 提示标题` + 缩进内容
