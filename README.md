# MP Preview

一键将 Obsidian 笔记转换为微信公众号格式的插件。

![version](https://img.shields.io/github/v/tag/rezonegame/mp-preview?color=blue&label=version&style=flat) ![license](https://img.shields.io/badge/license-MIT-green)

## 📦 基础功能
- 📝 **一键转换**：将 Markdown 文档快速转换为微信公众号兼容格式。
- 🎨 **丰富模板**：提供数十种精美模板（Minimal、Focus、Elegant 等系列），支持自定义字体与字号。
- 📋 **便捷复制**：一键 Pub 复制，直接粘贴至微信公众号后台编辑器，所见即所得。
- 🖼️ **实时预览**：支持自定义排版背景、版式结构，即时查看编辑效果。

## 🚀 高级功能
- 📸 **长图导出**：支持将文章整体导出为长图，便于跨平台多渠道分享。
- 💬 **对话气泡与图片画廊**：通过自定义代码块 (` ```dialogue ` 与 ` ```gallery `) 快速实现对话气泡布局与多图横向滚动展示。
- 🎯 **Callout 支持**：完整支持并解析 Obsidian 的 18 种 Callout，自动转化为自带色彩与图标的精美引用块。
- ✍️ **中文排版优化**：基于 pangu.js 自动在中英文之间添加空格，并支持智能引号转换（「」↔ ""）。

## ✨ 特色功能
- 🔒 **防打断锁定**：编辑超长文档时可锁定预览区，避免频繁渲染打断思路。
- ✏️ **所见即修改**：允许在预览模式下直接编辑和修改文案细节，且不影响底层 Markdown 原文件。
- 🔍 **SEO 隐藏文字**：支持在文章中插入对读者不可见、但可被搜索引擎索引抓取的隐藏关键词。
- 🌙 **纯净隔离模式**：预览区域完全与 Obsidian 主题隔离，始终保持公众号标准的浅色阅读环境。
