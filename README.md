# MP Preview

一键将 Obsidian 笔记转换为微信公众号格式的插件。

![version](https://img.shields.io/github/v/tag/rezonegame/mp-preview?color=blue&label=version&style=flat) ![license](https://img.shields.io/badge/license-MIT-green)

## 📦 基础功能
- 📝 **一键转换**：将 Markdown 文档快速转换为微信公众号兼容格式。
- 🎨 **丰富模板**：提供数十种精美模板（Minimal、Focus、Elegant 等系列），支持自定义字体与字号。
- 📋 **便捷复制**：一键 Pub 复制，直接粘贴至微信公众号后台编辑器，所见即所得。
- 🖼️ **实时预览**：支持自定义排版背景、版式结构，即时查看编辑效果。

## 🚀 高级功能
- 📸 **长图导出**：支持将文章整体导出为长图，便于跨平台多渠道分享。
- 💬 **对话气泡与图片画廊**：通过自定义代码块 (` ```dialogue ` 与 ` ```gallery `) 快速实现对话气泡布局与多图横向滚动展示。
- 🎯 **Callout 支持**：完整支持并解析 Obsidian 的 18 种 Callout，自动转化为自带色彩与图标的精美引用块。
- ✍️ **中文排版优化**：基于 pangu.js 自动在中英文之间添加空格，并支持智能引号转换（「」↔ ""）。

## ✨ 特色功能
- 🔒 **防打断锁定**：编辑超长文档时可锁定预览区，避免频繁渲染打断思路。
- ✏️ **所见即修改**：允许在预览模式下直接编辑和修改文案细节，且不影响底层 Markdown 原文件。
- 🔍 **SEO 隐藏文字**：支持在文章中插入对读者不可见、但可被搜索引擎索引抓取的隐藏关键词。
- 🌙 **纯净隔离模式**：预览区域完全与 Obsidian 主题隔离，始终保持公众号标准的浅色阅读环境。

## 📝 更新说明

### v2.0.9 (2026-05-14)

本次更新继续保持 **非 AI、本地排版增强** 的插件定位，不加入写作生成、意图识别或微信发布 API，重点增强公众号排版组件与主题库。

#### 新增
- **排版增强面板**：新增独立的「排版增强」设置区，可按需开启或关闭自动目录、任务列表增强、图片图注、表格优化、作者卡和关注引导。
- **本地排版组件**：新增 `toc`、`steps`、`checklist`、`quote-card`、`summary`、`author-card`、`subscribe`、`faq`、`timeline`、`comparison-table` 等代码块组件。
- **自动阅读导航**：可根据文章中的 H2/H3 标题数量自动生成阅读导航，默认阈值为 3 个标题。
- **任务列表增强**：支持将 Obsidian/Markdown 任务列表转换为更适合公众号阅读的检查清单卡片。
- **图片 Alt 图注**：支持将图片 alt 文本显示为图注，提升图文排版完整度。
- **表格移动端优化**：为表格增加横向滚动容器，减少公众号窄屏阅读时的溢出问题。
- **作者卡与关注引导**：支持在设置中配置作者信息、关注引导、二维码等内容，并在文章末尾自动插入。
- **场景情绪主题**：新增 5 个主题：秋日暖光、春日清新、深海静谧、赛博霓虹、苹果产品。

#### 优化
- 新增组件均只影响预览与复制结果，不会修改原始 Markdown 文件。
- 显式代码块组件优先于自动增强规则，便于用户精确控制排版。
- 新主题已接入三栏式主题画廊，并按风格自动归类。
- 保持现有 `dialogue`、`gallery`、Callout、FrontMatter 卡片、SEO 隐藏文字和长图导出能力兼容。

#### 使用示例

````markdown
```toc
01 | 先看问题 | 为什么公众号文章需要阅读导航
02 | 再看方法 | 如何用组件增强排版
03 | 最后发布 | 复制到公众号后台检查效果
```

```steps
01 | 写好正文 | 先完成内容，不依赖 AI 改写
02 | 添加组件 | 用本地排版块强化阅读路径
03 | 复制发布 | 粘贴到公众号后台继续微调
```
````
